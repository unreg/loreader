# loreader
Reader for LINUX.ORG.RU

Get source:
```
git clone https://github.com/unreg/loreader
cd loreader
yarn
```

Running
```
cd loreader
react-native run-android
```

Apk build
```
cd loreader/android
./gradlew assembleRelease
```


More info https://facebook.github.io/react-native/docs/getting-started
